# CircularLed
CircularLedger is an open-source blockchain infrastructure designed to improve transparency and accountability in decentralized waste-to-energy systems.

The platform will enable:

• Tracking of organic waste flows from households to treatment facilities
• Smart contract-based payments to waste collectors
• Real-time environmental impact dashboards
• Open digital infrastructure for circular economy systems

The project is currently at the concept stage and will develop a prototype with support from the UNICEF Venture Fund.

Lead organization: Waste Bangladesh
Website: https://wastebd.org
